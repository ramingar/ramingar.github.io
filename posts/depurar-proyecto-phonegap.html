Depurar un proyecto PhoneGap para Android

29 de diciembre de 2015

En el post anterior se comentó cómo iniciar un proyecto nuevo con PhoneGap. PhoneGap te proporciona un API para atacar al hardware de un dispositivo móvil usando javascript. En este post se hablará de cómo depurar el proyecto.

NOTA: se asume que

  • Apunta id de vendor de móvil -> http://developer.android.com/tools/device.html#VendorIds (BQ = 0x2a47) (OnePlus = idVendor: 05c6, idProduct: 6764)
  • Modifica o crea /etc/udev/rules.d/51-android.rules
  • Añade: SUBSYSTEM=="usb", ATTR{idVendor}=="0x2a47", MODE="0666", GROUP="plugdev"
  • Modifica: ~/.android/adb_usb.ini
  • Añade una línea: 0x2a47
  • adb kill-server
  • adb devices
